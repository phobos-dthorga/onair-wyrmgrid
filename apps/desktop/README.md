# WyrmGrid desktop

This package contains the Tauri 2 desktop shell and Svelte/MapLibre interface.
It is an adapter over `wyrmgrid-application`; business rules do not belong in
Tauri commands or Svelte event handlers.

From the repository root:

```powershell
npm run check
npm run build
npm run dev
```

See the root [development guide](../../docs/development.md) for prerequisites.
