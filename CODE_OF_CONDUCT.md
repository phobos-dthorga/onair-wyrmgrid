# Community code of conduct

WyrmGrid is intended to be a welcoming, technically curious community. Treat
other contributors with respect, assume good faith, critique ideas rather than
people, and make room for different levels of experience.

Harassment, discrimination, threats, doxxing, sexualized attention, deliberate
disruption, and publication of another person's private information are not
acceptable.

Project maintainers may edit or remove contributions and may temporarily or
permanently restrict participation when behavior threatens the community or
project. Reports may be made privately to the project owner through GitHub.

This code applies in repository spaces and when representing the project in
public. Enforcement decisions should prioritize safety, proportionality, and a
clear explanation to the affected parties where doing so is safe.
